# تقدير القيمة الإيجارية للعقارات الاستثمارية (نسخة داخلية)

**للاستخدام الداخلي – غير مخصص للنشر**

هذه الحزمة تجمع النظام الكامل + توسعة الذكاء الاصطناعي (مستوى 1+2+3) على شكل **زر اختياري** داخل النظام.
الذكاء الاصطناعي لا يغيّر أي حسابات أو قيود، ويعمل كتفسير/مقارنة/تحليل استراتيجي فقط.

## النشر
1) ارفع الملفات (بعد فك الضغط) إلى GitHub (Private).
2) Streamlit Cloud → New app → اختر repo → main → `app.py` → Deploy
3) ضع أسرار الدخول في Secrets (مستحسن):

```toml
ADMIN_USERNAME="admin"
ADMIN_PASSWORD="ضع_كلمة_قوية"
COOKIE_SECRET="نص_طويل_عشوائي"
```
